## Purpose

Defines the minimum visual contract for FSIPS exercise notation.

## Core Principle

The visual represents a playable instance, not the exercise definition.

## Required Visual Elements

- Ordered pitch sequence
- Rhythmic grid with pattern-derived time signatures
- Starting tempo (with future tempo progression support)

## Defaults

Neutral articulation, dynamics, phrasing, and register.

## Optional Elements

Fingering, string numbers, position markers, repeats.

## Explicit Exclusions

Technique markings, overload metadata, tuning diagrams, register fixation.

## Scope

Designed for warmups, scales, arpeggios; expanded later via versioning.
